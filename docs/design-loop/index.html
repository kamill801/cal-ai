Design Rules To Carry Into Figma

Use ranges, not false precision 640-760 kcal feels more honest than a single exact number.
Ask only one useful question The question must explain how it improves the estimate.
Separate facts from assumptions Confirmed items and guessed portions should be visually distinct.
Make correction feel primary Food rows need edit affordances before the record CTA.
